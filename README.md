A Multi-Page Full-Stack Social Media App for Developer Portfolios

- Developers can put their portfolio online with projects, contact and other information.
- Comment an issue and start discussion

Back End:

- Mongo DB
- Express
- Passport JS
- JWT Auth.

Front End:

- REACT
- Redux
- Bootstrap
- CSS
- HTML
- Material UI

Deployment

- Heroku
- AWS
- MongoDB
